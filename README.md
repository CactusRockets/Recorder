# Gravador de dados do monitor serial

Programa que serve para armazenar dados vindos da serial. Assim, todo o log da serial que é visualizado através dele é salvo em um arquivo TXT no mesmo diretório de execução do programa.

### Como usar

Para executar esse programa é necessário ter o [Processing IDE](https://processing.org/download) instalado no computador.

Após instalá-lo, abra o arquivo "Recorder.pde" usando o Programa. Perceba que algumas bibliotecas são necessárias para a execução do programa, são elas:

- processing.serial
- java.awt.Frame

OBS.: Se elas já não estiverem instaladas, pode-se instalar facilmente através do gerenciador de bibliotecas do próprio ProcessingIDE.

Após as devidas configurações, apertamos o botão de "Play" no Processing e o programa "Recorder.pde" deve ser executado.
